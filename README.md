# Projet-TCP

Implémentation d'un serveur/client permettant d'enregistrer un client et de déposer/retirer de l'argent. L'implémentation n'est pas optimale, on aurait souhaité avoir un fichier txt contenant toutes les informations afin de garder en mémoire les informations. De plus, l'inmplémentation ne gère pas les mauvaises saisies de l'utilisateur, cela peut causer des problèmes.
